Aliasing Input Properties

Es ponerle un Alias al input properties.

El alias se agrega como un string dentro del Decorator @Input('alias'), al momento de declara la propiedad.

Es util poner alias cuando por motivos de refactoring cambiamos los nombres de las variables. Al tener un alias el nombre queda igual en el componente padre donde estamos ingresando valores a esta propiedad, por ende son menos lugares donde tenemos que cambiar datos.
