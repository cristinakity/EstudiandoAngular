Input Properties

Son los datos de entrada o de State que tendra nuestro componente.

Para poder tulizar los input debemos importar @Input Decorator de @angular/core

En el sisguiente ejemplo las propiedades Name y Age se pasan al componente de la siguiente manera:

<input-properties [name]="'Juan'" [age]="20"></input-properties>
