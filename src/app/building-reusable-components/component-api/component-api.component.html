Component API

Básicamente es que cosa pueden enviarle a tu component y que cosas están expuestas de tu componente. Es como crear properties para tu componente, en un componente no se pueden usar poperties igual que en los HTML tags así que es necesario crearlas, a todo ese conjunto se le conoce como Component API.

Ejemplo:
< curso [isActive]="isActive" (change)="onCursoChange()"></curso>

Componente: Para hacer a un componente reutilizable necesitamos usar inputs y outputs properties.

  • Input: Usamos inputs properties para pasarle datos de entrada o de State al componente.
  • Output: Usamos output properties para lanzar eventos.

Input & Output properties son la API publica del componente.
