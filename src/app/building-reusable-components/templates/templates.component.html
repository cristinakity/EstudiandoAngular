Templates

Templates para los componentes. Template tecnicamente es la vista o View dentro de MVC, en angular template es codigo html en conjunto con angular code.

Existen dos maneras de usar templates en un componente.

  1. templateUrl: El template esta almacenado en un archivo externo. Se le asigna un string con la ruta de nuestro template .html
  2. template: Se agrega inline. Se puede usar un string '', o utlizar multilenas con las comillas invertidas ``

No es posible mezclar ambas propiedades del componente, se debe elejir templateUrl o template.

La eleccion que hagamos dependera del tipo de proyecto que estemos desarrollando, si el componente es algo muy simple y requiere un template simple es mejor utilizar inline template, en cambio si estamos desarrollando un componetne muy complejo sera mejor utilizar un template externo mdiante un archivo template html.
