Styles

Los styles son los estilos css que seran aplicados a cada elemento de nuestro compoenente. El estilo representa la parte visual del componente, que color de fondo, que tipo de letra, que tama√±o, que borde, etc. Existen muchas propiedades para cambiar dentro de style.

Existen tres maneras de usar styles en un componente.

  1. styleUrls: Se agrega un array de styles externos: ['./style1.css','./style2.css']
  2. styles: Se agrega inline. Se puede agregar un array de style multilena con las comillas invertidas `` y el contenido del style es como en un css.
    [`
    body {
        color: black;
        font-size: 14px;
    }`
    `
    p {
        color: gray;
        font-size: 13px;
    }`
    ]
  3. inline in the hmlt template: Igual como se hace en cualquier html web.
    <style>
    body {
        color: black;
        font-size: 14px;
    }
    </style>
    * Esto seria considerado una mala practica de programacion, pero es una de las cosas que angular nos permite hacer al momdento de definir styles para nuestro componente.

Es posible agregar styleUrl en conjunto con styles dentro de un componente, pero un estilo sobre-escribira a otro. El que se haya declarado al final del componente sera el que tome efecto.

Angular creara un scope para estos estilos, lo cual significa que solo funcionaran dentro del componente y no afectaran otros componentes.
