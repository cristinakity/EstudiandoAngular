Building a bootstrap form

Como crear un formulario utilizando los estilos de Bootstrap.

Para construir formularios con Bootstrap tendremos que utilizar la siguiente estructura:

  • div.group-form: Un div con la clase group-form
    • label[for="idInput"]: Un label con el atributo for seguido el id del input, esto hara que al darle click al label el foco se ponga en el input.
    • input.form-control[id="idInput"][type="text"]: Un input con la clase form-control esta hara el input mas "bonito" visualmente, el id deberia ser el mismo que en el for del label.
