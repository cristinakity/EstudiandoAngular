Custom Pipes

Son Pipes customizados o creados por el usuario.
Son formatos que se le puden aplicar a variables al momento de mostrarlas en la vista.
se utiliza el caracter pipe | seguido del tipo de Pipe que sera aplicado.

Pasos para crear un custom pipe:
  1. ng g p nombreDelPipe
  2. Se tiene que importar Pipe, PipeTranform de @angular/core ( al generarse con el comando esto se agrega automaticamente).
  3. En el metodo transform ponemos la funcion de nuestro Custom Pipe
  4. Agregar el pipe a app.module.ts
