ngIf

Sirve para cuando queremos mostrar u ocultar parte de una pagina dependiendo de una condicion.

Las directivas se usan para modificar el DOM

Hay dos tipos de Directives:
  1. Structural
    Modifican la estructura del DOM; Agregando o editando elementos del DOM.
    Se usa el prefijo * (asterisco).
  2. Attribute
    Modifican los atributos del los elementos del DOM.

En el primer ejemplo usamos:
<div *ngIf="btn1.isOn" class="alert alert-success">El boton 1 esta encendido</div>
<div *ngIf="!btn1.isOn" class="alert alert-danger">El boton 1 esta apagado</div>

En el segundo ejemplo utilizamos otra forma de agregar el *ngIf y ademas agreagmos una condicion else.

<div *ngIf="btn2.isOn then templateOn else templateOff"></div>
<ng-template #templateOn>
    <div class="alert alert-success">
        El boton 2 esta encendido
    </div>
</ng-template>
<ng-template #templateOff>
    <div class="alert alert-danger">
        El boton 2 esta apagado
    </div>
</ng-template>

Cuando el valor de ngIf es true el emento se agregara al DOM y cuando el valor es false el elemento se removera del DOM.

Utilizando templates y else en *ngIf tenemos un codigo mas legible, debido a que en el primer ejemplo donde teniamos dos divs con ngIf no podemos saber si uno esta actuando como else de otro, no sin tener que revisar el codigo y entender la logica aplicada; Por lo tanto utilizando ngIf con else es una forma mas clara de aplicar la logica necesaria y en este caso seria una mejor practica, igual nos ayudara si revisamos el codigo en el futuro y queremos entender que tratabamos de hacer.
