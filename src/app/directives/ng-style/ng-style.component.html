ngStyle

Es una directiva de atributos, osea que se utiliza para modificar atributos de los elementos del DOM. Sirve para agregar multiples estilos a un elemento.

A diferencia de [style.estiloDelDOM] property, [ngStyle] directive nos permite agregar multiples estilos, en el caso de la propiedad style tenemos que escribirla n veces cuando tenemos multiples estilos aplicadas a un elemento.
