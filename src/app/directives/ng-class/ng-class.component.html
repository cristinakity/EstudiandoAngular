ngClass

Es una directiva de atributos, osea que se utiliza para modificar atributos de los elementos del DOM. Sirve para agregar multiples clases a un elemento.

A diferencia de [class.className] property, [ngClass] directive nos permite agregar multiples clases, en el caso de la propiedad class tenemos que escribirla n veces cuando tenemos multiples clases aplicadas a un elemento.
