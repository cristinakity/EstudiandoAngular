ngFor

Sirve para iterar elementos en el DOM.

Se utiliza ngFor cuando tenemos una lista de objetos y queremos iterar la lista para mostrar todos los objetos.
ngFor cuenta con varios export values como index, que nos ayudan a tener algunas caracteristicas extras.
Podemos ver la lista de Local Variables para ngFor en el siguiente enlace.

Haciendo uso de *ngFor se enlistas los exports values:
  • {{ev.name}}: {{ev.type}}: {{ev.description}}
Promedio de alumnos
  • {{i}} - {{alumno.nombre}} : {{alumno.promedio}}
